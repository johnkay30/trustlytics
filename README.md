# Professional Portfolio Website

A modern, responsive portfolio website for IT Support and Data Analyst professionals.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI/UX**: Clean, professional design with smooth animations
- **Sections Included**:
  - Hero/Introduction section
  - About Me
  - Skills & Expertise (IT Support & Data Analysis)
  - Featured Projects
  - Professional Experience Timeline
  - Contact Form
- **Interactive Elements**:
  - Smooth scrolling navigation
  - Animated skill progress bars
  - Mobile-friendly hamburger menu
  - Form validation

## Getting Started

### Prerequisites

No special prerequisites needed! This is a static website that runs entirely in the browser.

### Installation

1. Clone or download this repository
2. Open `index.html` in your web browser
3. That's it! The website is ready to use.

### Customization

To personalize the portfolio:

1. **Update Personal Information**:
   - Open `index.html`
   - Replace "Your Name" with your actual name
   - Update email, phone, and location in the contact section
   - Add your social media links (LinkedIn, GitHub, etc.)

2. **Update About Section**:
   - Modify the about text to reflect your background
   - Update the statistics (projects, years of experience, clients)

3. **Customize Skills**:
   - Adjust skill names and percentages in the skills section
   - Add or remove tools and technologies

4. **Add Your Projects**:
   - Update project titles, descriptions, and tags
   - Add links to your actual project repositories or demos
   - Replace placeholder icons with project screenshots if desired

5. **Update Experience**:
   - Modify the timeline with your actual work experience
   - Update dates, company names, and job descriptions

6. **Styling**:
   - Modify `styles.css` to change colors, fonts, or layout
   - The color scheme uses CSS variables for easy customization

## Contact Form

The contact form is currently set up with client-side validation. To make it functional, you'll need to:

1. Set up a backend service (e.g., Formspree, EmailJS, or your own server)
2. Update the form submission handler in `script.js`
